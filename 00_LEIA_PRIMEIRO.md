## 🔧 CONFIGURAÇÃO RÁPIDA

### 1. Abra `.env` e configure:
```
DATABASE_URL="mysql://root:@localhost:3306/farmacia_db"
```

### 2. Crie o banco MySQL:
```sql
CREATE DATABASE farmacia_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 3. Rode:
```bash
npm run dev
```
